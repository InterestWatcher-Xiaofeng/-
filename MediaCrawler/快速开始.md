## 🎨 三大模块使用

### 模块1: 🔍 多关键词搜索模式 (推荐)

**适用场景:** 研究特定话题,批量采集数据

**操作步骤:**

```
第1步: 选择平台
  → 点击 "🎵 抖音"

第2步: 选择模式
  → 选择 "🔍 关键词搜索"

第3步: 输入关键词
  → 在文本框中输入 (每行一个):
     东方女孩
     程序员
     AI技术

第4步: 配置参数
  → 切换到"采集设置"标签页
  → 内容数量: 20 (每个关键词20个视频)
  → 评论数量: 100 (每个视频100条评论)

第5步: 登录平台
  → 切换到"登录管理"标签页
  → 点击"开始登录"
  → 扫码登录

第6步: 开始采集
  → 点击 "🚀 开始采集"
  → 等待完成

第7步: 查看结果
  → 切换到"结果查看"标签页
  → 点击"📁 打开数据目录"
```

**预期结果:**
```
3个关键词 × 20个视频 = 60个视频
60个视频 × 100条评论 = 6000条评论

输出文件:
data/douyin/csv/detail_comments_*.csv
```

## ❓ 常见问题

### Q1: 抖音关键词搜索返回空数据?

**A:** 这是正常的,抖音Search API被拦截了

**解决方案:**
- GUI会自动使用RPA混合模式
- 浏览器会自动打开
- 等待RPA搜索完成
- 不要手动关闭浏览器

---

### Q2: 如何获取创作者ID?

**A:** 从创作者主页链接提取

**示例:**
```
主页链接:
https://www.douyin.com/user/MS4wLjABAAAATJPY7LAlaa5X-c8uNdWkvz0jUGgpw4eeXIwu_8BhvqE

创作者ID:
MS4wLjABAAAATJPY7LAlaa5X-c8uNdWkvz0jUGgpw4eeXIwu_8BhvqE
```

---

### Q3: 如何批量输入视频链接?

**A:** 目前GUI只支持单个链接

**解决方案:**
- 使用模块1 (多关键词搜索) 自动收集链接
- 或手动修改配置文件 `config/dy_config.py`

```python
DY_SPECIFIED_ID_LIST = [
    "链接1",
    "链接2",
    "链接3",
    # ...
]
```

---

### Q4: 数据保存在哪里?

**A:** 默认保存在 `data/` 目录

**目录结构:**
```
data/
├── douyin/
│   ├── csv/
│   │   ├── detail_contents_*.csv
│   │   └── detail_comments_*.csv
│   └── json/
│       └── ...
├── xhs/
└── ...
```

**快速打开:**
- 点击"📁 打开数据目录"按钮

### Q5: 如何生成词云图?

**A:** 使用内置功能

**操作:**

---

### Q6: 采集速度慢怎么办?

**A:** 调整并发参数

**操作:**
```
1. 切换到"采集设置"标签页
2. 减少评论数量
3. 减少视频数量
4. 分批采集
```

**建议:**
```
保守配置:
- 视频数量: 10
- 评论数量: 50

平衡配置:
- 视频数量: 20
- 评论数量: 100

激进配置:
- 视频数量: 50
- 评论数量: 200
```

---

### Q7: 如何导出Excel格式?

**A:** CSV文件可以直接用Excel打开

**操作:**
```
1. 打开数据目录
2. 找到 detail_comments_*.csv
3. 右键 → 打开方式 → Excel
```

**或者:**
```
1. 打开Excel
2. 文件 → 打开
3. 选择 detail_comments_*.csv
4. 选择分隔符: 逗号
```
