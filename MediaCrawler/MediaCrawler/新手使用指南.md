## 第一步: 下载项目

### 方法1: 使用Git (推荐)

如果你已经安装了Git:

```bash
git clone https://github.com/InterestWatcher-Xiaofeng/MediaCrawler.git
cd MediaCrawler
```

### 方法2: 直接下载ZIP

1. 打开项目主页: https://github.com/InterestWatcher-Xiaofeng/MediaCrawler
2. 点击绿色的 `Code` 按钮
3. 选择 `Download ZIP`
4. 解压到任意文件夹

---

## 第二步: 安装Python环境

### 检查是否已安装Python

打开命令行(Windows按 `Win+R`,输入 `cmd`,回车),输入:

```bash
python --version
```

如果显示 `Python 3.9.x` 或更高版本,说明已安装,可以跳过此步骤。

### 安装Python (如果未安装)

1. 访问Python官网: https://www.python.org/downloads/
2. 下载 **Python 3.9** 或更高版本
3. 运行安装程序
4. ⚠️ **重要:** 勾选 `Add Python to PATH`
5. 点击 `Install Now`

### 验证安装

重新打开命令行,输入:

```bash
python --version
```

应该显示类似 `Python 3.9.13` 的版本号。

---

## 第三步: 安装依赖

### 3.1 进入项目目录

```bash
cd MediaCrawler
```

### 3.2 安装依赖包

```bash
pip install -r requirements.txt
```

⏳ **预计时间:** 3-5分钟 (取决于网络速度)

### 3.3 安装浏览器驱动

```bash
playwright install chromium
```

⏳ **预计时间:** 2-3分钟

### 3.4 验证安装

如果没有报错,说明安装成功!

---

## 第四步: 启动GUI

### Windows用户

**方法1: 双击启动 (最简单)**

1. 找到项目文件夹中的 `启动GUI.bat` 文件
2. 双击运行
3. 等待几秒钟,GUI窗口会自动弹出

**方法2: 命令行启动**

```bash
python start_gui.py
```

### Mac/Linux用户

```bash
python start_gui.py
```

### 启动成功标志

- 弹出一个GUI窗口
- 窗口标题为 "抖音评论采集工具"
- 有三个标签页: "采集设置", "数据输出", "关于"

---

## 第五步: 登录抖音

> ⚠️ **重要:** 必须先登录抖音,才能开始采集!

### 5.1 点击"登录抖音"按钮

1. 在GUI界面顶部,找到 `登录抖音` 按钮
2. 点击按钮

### 5.2 等待浏览器弹出

- 会自动打开一个Chrome浏览器窗口
- 浏览器会自动跳转到抖音登录页面

### 5.3 扫码登录

1. 打开手机抖音APP
2. 点击右下角 `我`
3. 点击右上角 `≡` (三条横线)
4. 点击 `扫一扫`
5. 扫描电脑屏幕上的二维码
6. 在手机上确认登录

### 5.4 确认登录成功

- 浏览器页面显示你的抖音主页
- GUI界面显示 `✅ 已登录`
- 终端显示 `✅ 登录任务完成`

### 5.5 关闭浏览器

- **不要手动关闭浏览器!**
- 浏览器会在后台保持运行
- 这是为了保持登录状态

## 第六步: 开始采集

### 6.1 选择采集模式

GUI提供三种采集模式:

#### 模式1: 关键词搜索 🔍

**适用场景:** 想要采集某个话题/关键词下的视频评论

**操作步骤:**
1. 选择 `关键词搜索` 单选按钮
2. 在输入框中输入关键词,例如: `美食`
3. 设置采集数量:
   - 视频数量: `10` (采集10个视频)
   - 评论数量: `50` (每个视频采集50条评论)
4. 点击 `开始采集`

**示例:**
```
关键词: 美食
视频数量: 10
评论数量: 50

结果: 采集10个关于"美食"的视频,每个视频50条评论,共500条评论
```

---

#### 模式2: 指定内容详情 🎬

**适用场景:** 想要采集特定视频的评论

**操作步骤:**
1. 选择 `指定内容详情` 单选按钮
2. 在输入框中输入视频链接,每行一个:
   ```
   https://www.douyin.com/video/7525538910311632128
   https://www.douyin.com/video/7202432992642387233
   https://www.douyin.com/video/7270814012802174265
   ```
3. 设置评论数量: `100` (每个视频采集100条评论)
4. 点击 `开始采集`

**如何获取视频链接?**
1. 打开抖音APP或网页版
2. 找到想要采集的视频
3. 点击 `分享` 按钮
4. 选择 `复制链接`
5. 粘贴到输入框中

**支持的链接格式:**
- 完整链接: `https://www.douyin.com/video/7525538910311632128`
- 短链接: `https://v.douyin.com/drIPtQ_WPWY/`
- 纯ID: `7525538910311632128`

**输出结果:**
- 文件名: `20251103_182500_3条视频_评论.csv`
- 内容: 所有3个视频的评论都在这一个文件中

---

#### 模式3: 创作者主页 👤

**适用场景:** 想要采集某个博主所有视频的评论

**操作步骤:**
1. 选择 `创作者主页` 单选按钮
2. 在输入框中输入创作者主页链接:
   ```
   https://www.douyin.com/user/MS4wLjABAAAAf60hibYyjDzX1uR6Kei2JglOo0FawxUCWh5chQJ-Hd7f3StJQ-LX0f263D1DWQor
   ```
3. 设置采集数量:
   - 视频数量: `20` (采集该博主的20个视频)
   - 评论数量: `50` (每个视频采集50条评论)
4. 点击 `开始采集`

**如何获取创作者主页链接?**
1. 打开抖音APP或网页版
2. 进入博主的主页
3. 点击 `分享` 按钮
4. 选择 `复制链接`
5. 粘贴到输入框中

**输出结果:**
- 文件名: `博主昵称_20条视频_评论.csv`
- 例如: `美食博主张三_20条视频_评论.csv`
- 内容: 该博主20个视频的所有评论

---

### 6.2 配置输出设置

切换到 `数据输出` 标签页:

#### 输出格式

- **CSV文件 (推荐):** Excel可以直接打开,方便查看和分析
- **JSON文件:** 适合程序员进一步处理

#### 输出目录

- 默认: `./data` (项目文件夹下的data文件夹)
- 自定义: 点击 `浏览` 按钮选择其他文件夹

#### 评论设置

- **启用评论采集:** 必须勾选,否则不会采集评论
- **启用二级评论:** 勾选后会采集评论的回复(最多100条)

---

## 常见问题

### Q1: 启动GUI时报错 "No module named 'xxx'"

**原因:** 依赖包没有安装完整

**解决方法:**
```bash
pip install -r requirements.txt
```

---

### Q2: 登录时浏览器没有弹出

**原因:** Playwright浏览器驱动没有安装

**解决方法:**
```bash
playwright install chromium
```

---

### Q3: 采集时显示"采集过程中出错"

**可能原因:**
1. 没有登录抖音
2. 登录已过期
3. 网络连接问题

**解决方法:**
1. 重新登录抖音
2. 检查网络连接
3. 查看终端输出,确认具体错误信息

---

### Q4: 采集速度很慢

**原因:** 为了避免被抖音反爬虫检测,程序会自动限速

**正常速度:**
- 每个视频采集时间: 30秒 - 2分钟
- 10个视频预计时间: 5-20分钟

**不要:**
- ❌ 不要同时运行多个采集任务
- ❌ 不要频繁重启程序
- ❌ 不要修改延迟时间(可能导致账号被封)

---

### Q5: CSV文件乱码

**原因:** Excel默认编码问题

**解决方法:**
1. 用记事本打开CSV文件
2. 点击 `文件` -> `另存为`
3. 编码选择 `UTF-8`
4. 保存后用Excel打开

或者直接用WPS打开,WPS会自动识别编码。

---

### Q6: 想要采集更多评论

**限制说明:**
- 单个视频最多采集1000条评论(抖音API限制)
- 二级评论最多100条(程序限制,可修改)

**如何修改:**
1. 在GUI中设置评论数量: `1000`
2. 如果需要更多,需要修改源代码

---

### Q7: 如何停止采集?

**方法1: 点击停止按钮**
1. 在GUI界面点击 `停止采集` 按钮
2. 等待当前任务完成
3. 已采集的数据会自动保存

**方法2: 强制停止**
1. 关闭GUI窗口
2. 关闭终端窗口
3. ⚠️ 注意: 当前正在采集的数据可能丢失

---

### Q8: 登录状态能保持多久?

**保持时间:** 通常7-30天

**失效标志:**
- 采集时提示"需要登录"
- 浏览器自动跳转到登录页面

**解决方法:**
- 重新点击 `登录抖音` 按钮
- 重新扫码登录

---

### Q9: 可以同时采集多个平台吗?

**当前版本:** ❌ 不支持

**原因:** 本项目是基于 [NanmiCoder/MediaCrawler](https://github.com/NanmiCoder/MediaCrawler) 的二次开发版本,目前只开放了抖音的三大功能:
1. 关键词搜索
2. 指定内容详情
3. 创作者主页

**其他平台:** 小红书、B站、快手等平台的功能已被禁用,后续版本可能会开放。
