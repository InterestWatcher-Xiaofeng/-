## 🚀 使用说明

### 启动软件

双击运行：`红枫工具箱/启动红枫工具箱.bat`

或直接运行：`红枫工具箱/红枫工具箱.exe`

### 首次使用

1. **选择平台**：选择"抖音"
2. **登录账号**：
   - 点击"登录账号"按钮
   - 在弹出的浏览器中扫码登录
   - 完成手机号验证（如果需要）
   - 等待登录成功提示

3. **配置采集参数**：
   - 输入视频链接（每行一个）
   - 设置最大评论数（建议500-1000）
   - 选择输出目录

4. **开始采集**：
   - 点击"开始采集"按钮
   - 观察进度条实时更新
   - 等待采集完成

### 登录验证提示

如果采集过程中弹出验证提示：

```
检测到可能需要登录验证！

请在浏览器中完成以下操作：
1. 扫码登录（如果需要）
2. 完成手机号验证（如果需要）
3. 完成滑动验证码（如果需要）

验证完成后，采集将自动继续。
此过程中点击【停止采集】按钮无效。
```

**处理方法**：
- 在浏览器中完成相应的验证操作
- 等待60秒后，采集会自动继续
- 不要点击"停止采集"按钮

## 📋 采集数据说明

### 输出格式

采集的数据会保存为CSV文件，包含以下字段：

| 字段名 | 说明 |
|--------|------|
| `视频链接` | 视频的完整URL |
| `视频标题` | 视频的标题 |
| `评论内容` | 评论的文字内容 |
| `评论者昵称` | 发表评论的用户昵称 |
| `评论者IP` | 评论者的IP属地 |
| `点赞数` | 评论的点赞数量 |
| `评论时间` | 评论发表的时间 |
| `子评论数量` | 该评论的回复数量 |
| `评论ID` | 评论的唯一标识符 |

### 文件位置

默认保存在：`红枫工具箱/data/抖音/` 目录下

文件名格式：`抖音_评论_YYYYMMDD_HHMMSS.csv`

## ⚙️ 配置说明

### 采集参数

- **最大评论数**：每个视频最多采集的评论数量
  - 建议值：500-1000
  - 过大可能导致采集时间过长
  - 过小可能无法获取足够数据

- **输出目录**：数据保存的位置
  - 默认：软件目录下的`data`文件夹
  - 可自定义任意目录

### 浏览器数据

登录信息保存在：`红枫工具箱/browser_data/` 目录

**注意**：
- ✅ 不要删除此目录，否则需要重新登录
- ✅ 此目录包含登录凭证，请妥善保管
- ✅ 重新安装软件时，可保留此目录以保持登录状态

## 🐛 常见问题

### Q1: 软件无法启动？

**解决方法**：
1. 确认系统是Windows 10/11 (64位)
2. 关闭杀毒软件或添加信任
3. 以管理员身份运行

### Q2: 登录后仍提示需要验证？

**解决方法**：
1. 完成浏览器中的所有验证步骤
2. 等待60秒后自动继续
3. 如果仍失败，删除`browser_data`目录重新登录

### Q3: 采集速度很慢？

**解决方法**：
1. 检查网络连接
2. 减少最大评论数
3. 避免同时采集过多视频

### Q4: 采集到的评论数量少于设置值？

**原因**：
- 视频本身的评论数量不足
- 部分评论被隐藏或删除
- 平台限制了评论的可见性

### Q5: 进度条不更新？

**解决方法**：
1. 等待几秒，进度条会延迟更新
2. 检查是否正在进行登录验证
3. 重启软件重试
